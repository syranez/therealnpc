Open the editor
---------------

1. vim

        vim

on most systems you can vi also to start vim.

2. emacs

        emacs

Open the file "foobar.txt"
--------------------------

1. vim

        (ESC) :e foobar.txt

2. emacs

        C-x C-f foobar.txt


Save the file as "foobar2.txt"
------------------------------

1. vim

        (ESC) :w foobar2.txt

1. emacs

        C-x C-w foobar2.txt

Exit the editor
---------------

1. vim

        (ESC) :q

1. emacs

        C-x C-c
