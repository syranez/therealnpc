Deletion
========

Here are delete challenges.

Delete the character under curser
---------------------------------

1. vim

        (ESC) x

1. emacs

        C-d

Delete the next n characters
----------------------------

1. vim

        (ESC) Nx

1. emacs

        M-N C-d

(where N is a number)

Delete the line
---------------

1. vim

        (ESC) dd

1. emacs

        C-S-<Backspace>

Delete from cursor position till end of paragraph
-------------------------------------------------

1. vim

        (ESC) d}

1. emacs

        M-x kill-paragraph

Delete from start of paragraph till cursor position
---------------------------------------------------

1. vim

        (ESC) d{
