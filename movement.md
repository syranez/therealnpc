Movement
========

Here are movement challenges.

Go to Nth line of buffer
--------------------------

1. vim

        ???

1. emacs

        M-g M-g N RET

(where N is a number)
        

Go to bottom of buffer
----------------------

1. vim

        G

1. emacs

        M->

Go to first line of buffer
--------------------------

1. vim

        gg

1. emacs

        M-<
or
        M-g M-g 0 RET
        
Go N words forward
------------------

1. vim

        ???

2. Emacs

        M-N M-f
        
(where N is a number)

Go N words backward
------------------

1. vim

        ???

2. Emacs

        M-N M-b
        
(where N is a number)
